Vision ML Tools

This plugin adds several tools and filters which use Neural Networks.

  • Segmentation Tools: Select objects by clicking on them or drawing a rectangle around them. Find them among the selection tools in the tool bar.
  • Inpainting Tool: Similar to Krita's "Smart Patch", uses a simple GAN to predict content to fill small gaps. The tool is located next to Smart Patch on the tool bar.
  • Background Removal: Filter which separates foreground objects from background. Can be found in the top menu below Filters > Other > Background Removal...
